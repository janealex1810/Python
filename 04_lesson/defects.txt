 #1: Неправильная капитализация первой буквы
- Описание: Метод `capitalize` неверно обрабатывает строки с уже заглавной первой буквой.
- Шаги:
  1. Создайте объект класса `StringUtils`.
  2. Вызовите метод `capitalize` с передачей строки "Already Capitalized".
- ОР: "Already capitalized"
- ФР: "Already capitalized" (результат соответствует, но необходимо уточнить логику)
- Приоритет: Низкий

 #2: Метод `trim` не удаляет пробелы перед строкой
- Описание: Метод `trim` не удаляет пробелы, если строка состоит только из пробелов.
- Шаги:
  1. Создайте объект класса `StringUtils`.
  2. Вызовите метод `trim` с передачей строки "   ".
- ОР: ""
- ФР:: ""
- Приоритет: Низкий

#3: Метод `contains` неверно определяет наличие символа
- Описание: Метод `contains` не учитывает регистр символов.
- Шаги:
  1. Создайте объект класса `StringUtils`.
  2. Вызовите метод `contains` с строкой "SkyPro" и символом "s".
- ОР: False
- ФР: True
- Приоритет: Высокий

#4: Неправильное удаление символов в методе `delete_symbol`
- Описание: Метод `delete_symbol` некорректно удаляет символы, если они встречаются многократно.
- Шаги:
  1. Создайте объект класса `StringUtils`.
  2. Вызовите метод `delete_symbol` с передачей строки "banana" и символа "a".
- ОР: "bnn"
- ФР: "bn"
- Приоритет: Высокий

#5: Метод `delete_symbol` возвращает строку без изменений при удалении несуществующего символа
- Описание: Метод `delete_symbol` должен возвращать оригинальную строку, если символ для удаления не найден, но неверно обрабатывает случаи с пробелами.
- Шаги:
  1. Создайте объект класса `StringUtils`.
  2. Вызовите метод `delete_symbol` с передачей строки "hello world" и символа "z".
- ОР: "hello world"
- ФР: "hello world" (результат соответствует, но нужно добавить сообщение, что символ не найден)
- Приоритет: Средний
